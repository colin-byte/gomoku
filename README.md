# 五子棋 (Gomoku)

终端版五子棋双人对战游戏

## 功能

- 15x15 标准棋盘
- 双人对战模式
- 自动判定胜负（连成五子获胜）
- 平局判定（棋盘下满）
- 重新开始功能
- 退出游戏 (输入 q)

## 运行方法

```bash
python main.py
```

## 操作说明

1. 输入坐标格式: `x y` (例如: `7 7` 表示棋盘中心天元)
2. x 表示列号（横向），y 表示行号（纵向）
3. x 和 y 的取值范围: 0-14
4. 黑方先手
5. 率先连成五子的一方获胜
6. 棋盘下满则平局
7. 游戏结束后可选择重新开始
8. 输入 `q` 退出游戏
